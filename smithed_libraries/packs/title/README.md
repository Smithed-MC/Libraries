# Title
This library helps multiple data packs manage the title / subtitle by aggregating usage by priority. This allows persistent messages to display without interrupting more urgent notification type messages.

## Usage
For usage information, see the [documentation](https://docs.smithed.dev/libraries/title)
## Downloading
You can download this library from [the website](https://smithed.net/packs/title)  
or  
You can build it from source using [beet](https://github.com/mcbeet/beet)

### Building
```
$ pip install beet mecha
$ git clone https://github.com/Smithed-MC/Libraries
$ cd Libraries/title
$ beet -s 'meta.libraries = ["title"]'
```
